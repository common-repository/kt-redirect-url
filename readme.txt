== Description ==

It is unique (Not developed before) in wordpress plugin life. This plugin created for redirecting URLs to specified Link under plugin setting. This plugin is most useful for those who are using premium themes( Free themes as well) and while importing demo content theme creates Unusual posts, pages, menus and more, in such cases site owner/ developer does't want to remove them permanantly but it headache for google ranking so to avoid such "404 & unuseful posts having dummy contents" and theme data to keep safe and rank over google.

= Features =

*  Extremely easy to use

*  No burden on the database

*  One step you to go here and avoid 404, dummy URLs

*  Improve ranking over SEO by eliminating dummy pages under dashboard to comes under google's crawl

*  Redirect all URL/Links which are not listed under posts/menu dropdown on "Redirect ALL" Setting page.

== Installation ==

1. Upload kt-redirect-url folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting "Redirection ALL" and select menus & posts which are not to be redirected as well as don't forget to set "Enable" option.

== How to use ==

1. Go to "Redirect ALL" Setting page in Dashboard.
2. Fill up the setting having: 
   * URL where the pages to be redirected, Select the menu under which pages will not be redirected i.e. Exclude from redirection that means the valid URLs on the sites to be included in the menu, 
   * Select the posts which are again valid and not to redirect(Exclude from redirection), 
   * and at the last use Redirection status as "Enable" to active the functionality of plugin,
   * If while any problem or temporarily halt regarding redirections just select the redirection status as "Disable" so you do not needs to deactivate or delete plugin.
